# Decode-Labs
Data Cleaning and Preparation Project
Project Overview
This project focuses on cleaning and preparing raw data for analysis using Microsoft Excel. The dataset was processed to improve data quality and ensure accurate analysis.
Objectives
Remove duplicate records
Handle missing values
Correct data formats
Standardize data entries
Prepare data for further analysis
Tools Used
Microsoft Excel
Data Cleaning Steps
Removed duplicate records.
Checked and handled missing values.
Applied appropriate data formats (Date, Number, Text).
Corrected inconsistent entries.
Verified data quality and accuracy.
Results
Clean and organized dataset
Improved data consistency
Ready for Exploratory Data Analysis (EDA)
